# Code of Conduct

AniGate aims to be a focused, practical project space.

## Expected Behavior

- Be respectful and direct.
- Critique code, designs, and risks without personal attacks.
- Keep discussions relevant to the issue or pull request.
- Assume good intent, but be precise about security and safety concerns.
- Disclose conflicts of interest when they affect a technical decision.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private information without permission.
- Posting secrets, tokens, credentials, or private repository content.
- Repeated off-topic or bad-faith comments.
- Pressuring maintainers to support commercial use outside the project license.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block accounts; or
decline contributions that violate this code.

If you need to report a conduct issue, open a minimal issue asking for a private
maintainer contact. Do not include sensitive personal details in a public issue.
